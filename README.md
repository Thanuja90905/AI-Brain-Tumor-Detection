# AI Brain Tumor Detection

## Description
This project detects brain tumors from MRI images using Deep Learning.

## Features
- Upload MRI scan
- Predict tumor presence
- Simple web interface

## Technologies Used
Python, TensorFlow, Flask

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000
